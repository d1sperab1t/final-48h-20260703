## 3. 游戏流程（mission.html）

```
Card 01: 抵达集结
  │  打开补给包，找到专属任务卡
  ↓
Gate 2: 输入神秘代码
  │  6位字母数字代码 → 识别玩家身份
  ↓
Card 02: 集结队友
  │  显示欢迎信息、搭子提示
  │  14:30 前抵达酒店大厅
  ↓
Gate 3: 输入队友神秘代码
  │  找到你的剧本杀搭子
  │  solo：输入自己的代码
  │  双人：输入队友代码
  │  三人：输入两位队友代码
  ↓
Card 03: 搭子揭晓
  │  显示搭子昵称
  ↓
Gate 4: 输入暗号
  │  "我是主理人，你是传承人"
  ↓
Card 04: 寻找传承人
  │  显示传承人姓名、地址
  │  任务：喊暗号 → 听故事 → 问关键词 → 换地图碎片
  ↓
Gate 5: 输入队友生日组合
  │  同组生日拼接（8/16/24位）
  ↓
Card 05: 探索莫干山户外运动协会
  │  3个问题 → 现场寻找答案
  ↓
终极密码: 输入终极密码
  │  "1901MGS"
  ↓
Card 06: 恭喜！传承人任务完成！
  │  领取路线关键词卡，设计专属路线
```

## 5. Gate 验证逻辑

### Gate 2 — 神秘代码识别

```
输入 → toUpperCase() → PARTICIPANTS[code] 查找
匹配 → 显示 Card 02 + Gate 3
不匹配 → 错误提示
```

### Gate 3 — 搭子验证

```
solo（SOLO_CODES 包含）→ 输入自己代码
双人（PAIRS 返回1个）→ 输入队友代码
三人（PAIRS 返回2个）→ 输入两位队友代码（两个输入框）
匹配 → 显示 Card 03 + Gate 4
不匹配 → 错误提示
```

### Gate 4 — 暗号验证

```
输入 === "我是主理人，你是传承人"
匹配 → 显示 Card 04（传承人地址）+ Gate 5
不匹配 → 错误提示
```

### Gate 5 — 生日组合验证

```
输入 === BIRTHDAY_PASSWORDS[当前玩家代码]
匹配 → 显示 Card 05（户外协会问题）
不匹配 → 错误提示
```

### 终极密码 — 最终解锁

```
输入 === "1901MGS"
匹配 → 隐藏终极密码面板 → 显示 Card 06（完成）
不匹配 → 错误提示
```

### 后门机制

```
Card 05 第一个输入框输入"莫干山" → 直接通过全部验证
```
