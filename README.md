# 莫干山 48H Break Point / Monster Speed

这一版根据用户提供的运动视觉参考重新调整：跑团海报、运动模糊、巨大白色标题、颗粒质感、速度切片字和更强的交互律动。

## 文件

- `invitation.html`：Break Point 风格入口页，运动模糊抽象背景、大白标题、点击波纹、横向跑动字幕、Canvas 速度线。
- `poster.html`：Monster Speed 风格任务页，保留倒计时和解锁逻辑，加入节拍条、点击波纹、任务卡翻转、行程高亮。
- `netlify.toml`：部署时把根路径指向 `invitation.html`。

## 视觉方向

- 参考 Break Point：居中大白字、运动模糊、颗粒海报。
- 参考 Monster Speed：速度切片字、强对比暗背景、微型信息组件。
- 参考 Beyond Number One：运动大片排版、巨型标题、底部 credits 信息系统。

## 互动

- 背景速度线持续运动。
- 点击页面产生圆形波纹。
- 任务卡点击翻转。
- 行程行点击高亮。
- 倒计时和节拍条有律动动画。
